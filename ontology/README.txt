SDTemplate 

- the core of the syndromic surveillance ontology
- contains top-level superclasses and individual syndrome classes, 
  but not specific conditions, concepts or keywords
- used by ontobuilder as template
- .owl + .pprj 
  .pprj file can be used to edit the template in Protege (3.4.1)

SyndromeDefinitions

- populated ontology generated by ontobuilder based on the template
- reflects current version of the template and the input spreadsheet 
  with clinical concepts
- .owl + .pprj

SyndromeDefinitions_vX.X.owl

- versioned snapshot of the Syndromic Surveillance Ontology
